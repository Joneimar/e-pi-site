Contexto

O que é o E-PI?

Com a Portaria Normativa nº 50/2022 do MME, todos os consumidores do Grupo A (média e alta tensão) passaram a poder migrar para o Ambiente de Contratação Livre (ACL). Isso representa potencial economia, mas exige análise técnica complexa.

O E-PI calcula o preço máximo da energia no ACL que equaliza os custos totais com o ACR — o Preço de Indiferença. Qualquer preço de mercado inferior a este valor indica vantagem econômica na migração.

Diferentemente das ferramentas comerciais do mercado, o E-PI é gratuito, transparente e sem viés comercial, desenvolvido em ambiente acadêmico para auxiliar na tomada de decisão dos consumidores.

O E-PI é uma evolução do software original registrado, com novos módulos de projeção, validação e interface web moderna.

Aplicação Web

Simulador Streamlit

Interface interativa desenvolvida em Python com a biblioteca Streamlit, permitindo que consumidores de qualquer localidade do Brasil simulem a viabilidade de migração.

Simulação Interativa

Insira dados da UC, histórico de consumo, tributos e parâmetros do ACL. Visualize o E-PI mensal e total em gráficos interativos.

Dados da ANEEL

Coleta automatizada de tarifas via API da ANEEL. Banco SQLite com atualização diária por distribuidora.

Projeções Futuras

Naive Sazonal para consumo, projeção tarifária, mudança de patamar de carga e horizonte contratual configurável.

Exportação

Resultados exportáveis em CSV e JSON para análises complementares ou apresentação a gestores.

Integração

Inserção no MACDE

O E-PI está sendo integrado à plataforma MACDE — Modelo de Avaliação dos Contratos de Demanda de Energia.

O MACDE é uma plataforma desenvolvida pelo Grupo de Estudos em Sistemas de Energia (GESE) do IFSC desde 2017, com apoio da Setec/MEC através do programa EnergIF.

A plataforma auxilia consumidores do Grupo A na otimização econômica de contratos de demanda e faz parte do Programa Conecta Rede (MEC/RNP), sendo acessível por toda a rede federal de educação profissional e tecnológica.

Na versão MACDE, o E-PI utiliza backend Python/Flask com API REST e frontend Vue.js, reaproveitando dados já coletados pela plataforma (tarifas, consumo, previsão) como parâmetros de entrada.
